# How to execute

<br>

## using yarn

<br>

install dependencies

```bash
yarn
```

runs the app in development mode

```bash
yarn start
```

run tests

```bash
yarn test
```

<br />

## using npm

<br />

install dependencies

```bash
npm i
```

runs the app in development mode

```bash
npm run start
```

run tests

```bash
npm run test
```
